### 本科生毕业设计 基于Thinkphp5的网上订餐系统

------

##### 后端项目启动步骤

使用 back/graduation.sql 中的 sql 语句创建数据库与数据库表

##### 前端项目启动步骤

进入front目录执行
~~~javascript
npm install
~~~

然后运行
~~~
npm run dev
~~~
