## 毕业设计 网上订餐系统（后端基于Thinkphp5）

------

##### 后端项目启动步骤

使用 back/graduation.sql 中的 sql 语句创建数据库与数据库表 后台账号密码  admin/123456

##### 前端项目启动步骤

进入front目录执行
~~~
npm install
~~~

然后运行
~~~
npm run dev
~~~
